<STYLE>

QUOTES [CRITICAL]
   Quote marks: " " only (no « »). Nested → ASCII '. Terminal punct outside.

EZAFE [STRONG] silent ه → هٔ (خانهٔ، درجهٔ). Never ه‌ی. ZWNJ for: present/past/negation prefixes (می‌، نمی‌)، suffixes (ها، تر، ترین)، established compounds (هم‌خانه، هم‌ساکن، دست‌وپاگیر، به‌شمار، بی‌خانمان). Avoid decorative or post-preposition ZWNJ.
HARAKAT [STRONG] strip all. Permit only homograph + unresolvable by ±5 context. Ceiling/100 lines: NEWS/FACILITATOR ≤3 | SAGE ≤5. Never on SMTV/proper names.
PUNCT [STRONG] No Oxford comma. Apposition dash " - ". SMTV compound "[H]- [S]". "؛" FORBIDDEN outside quote → "،" or clause split within SAME line. Inside verbatim/literary quote → byte-id.

NUMBERS [STRONG] VALUE-LOCKED + FORMAT-NORMALIZABLE.
   LOCKED: value, sign, range, percent, date, specificity, ordering.
   FORMAT:
     Digits → Persian ۰–۹. Thousands → "٬". Decimal → ".".
     ROUNDING — applies only to QUANTITY numbers. NEVER apply to: years (2026), IDs/codes (F-16, S01E05, H5N1), episode numbers (قسمت N از M), legal/article numbers, addresses, postcodes, timestamps (00:01:23,456), model/serial numbers, phone numbers.
       Apply MILLION rule before THOUSAND rule.
       Exact integer multiple of 1,000,000 → "n میلیون" (2,000,000 → ۲ میلیون).
       Else exact integer multiple of 1,000 → "n هزار" (3,000→۳ هزار | 15,000→۱۵ هزار | 25,000→۲۵ هزار). Special: 1,000 → "هزار" (without "۱").
       Non-round (1,500/3,750/12,400) → Persian digits with "٬"; never invent rounding.
     FORBIDDEN: "۱۰۰۰ هزار" | "۱ میلیون هزار" | "۱۰ هزار هزار".
     Percent: % → ٪.
     Date: D Month YYYY → "۴ مارس ۲۰۲۶".
     Range: a-b → "a تا b".
     Serial: Part N of M → قسمت N از M (قسمت locked, not بخش/پارت).
     Units: always translate with ezafe (°C → درجهٔ سلسیوس | km → کیلومتر). Never TECH_LOCK.
     Imperial: metric+imperial both → keep metric only. Imperial only → transliterate unit, keep value (5 miles → ۵ مایل). Never invent conversion.
</STYLE>

   a) Transferable image → structural translation preserving image.
      (not the sharpest knife in the drawer → تیزترین چاقوی کشو هم نیست)
   b) Foreign/misleading image → FA idiom, same weight.
      (break a leg → موفق باشی | spill the beans → لو دادن)
   c) No equivalent → meaning only. LAST RESORT.
   Wordplay/puns: preserve comic intent via verb/adjective; never flatten.
⑥ Recast for fluency: complex EN → ONE flowing FA line within boundary. Never glue short sentences.
⑦ Final gate:
   Persian SOV. Subject-to-verb distance ≤12 words.
   No "توسط". Bureaucratic compound → simple (مورد بررسی قرار داد → بررسی کرد).
   Speech verbs by context:
     declared → اعلام کرد
     warned → هشدار داد
     noted / remarked → خاطرنشان کرد
     emphasised / insisted → تأکید کرد
     urged + person + to do → از ... خواست (که) ...
     urged (public appeal) → فراخواند / خواستار شد
     urged (emphasis on necessity) → تأکید کرد
     Master speech:
       Master says → می‌فرمایند
       Master said → فرمودند
       Master has said → فرموده‌اند
   Soft targets (never break LINE): NEWS ≤12 words | SAGE/FACILITATOR ≤9 words.
   One stable meaning unit per line. Avoid bare-verb starts.
