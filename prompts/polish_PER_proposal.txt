<EDIT>
Run on flagged lines only. Priority order:

① Persianise English leakage (respect HARDLOCK HL-6 exceptions first):
   AI → هوش مصنوعی | EU → اتحادیه اروپا | UN → سازمان ملل | DNA → دی‌ان‌ای
   Known acronym → FA form. Unknown → Persian phonetic transliteration.
   EXCEPTION: HL-6 W3 codes (GPT-4o, F-16, H5N1) stay byte-id.

② SOV repair — move the verb to line-end, within the current PREDICATE line only.
   Fragment lines: skip. Verb already at line-end → CLEAN.

③ Break unnatural noun chains → verbal / prepositional structures (same line).

④ Replace calques with native Persian — only when you are certain.

⑤ Remove "توسط" → active voice, same line. Agentless passive permitted only in SAGE
   when no natural active form exists.

⑥ NATIVE_REGISTER repairs — ONLY when [FA] still carries visible EN-shape
   despite the translator's pass. This phase is RESIDUE-PATCHING, not a
   second translator pass. G9 (HANDOFF_TRUST) is the entry gate: if the line
   already reads native, skip ⑥ entirely.

   a) NATIVE_SYNTAX: if a line preserves the EN clause order
      ("X, با Y, Z را می‌کند"), restructure to Persian-first.
      ✗ "رابطه‌اش با دوست‌دخترش مگان، از راه عشق مشترکشان به اشخاص-حیوان، عمیق‌تر می‌شود."
      ✓ "عشق مشترک او و مگان به اشخاص-حیوان، رابطه‌شان را عمیق‌تر می‌کند."

   b) DOWNGRADE_LITERARY: replace elevated noun / verb choices with
      the everyday broadcast form, unless the source is poetic.
      ✗ "در ژرفای خود"   → ✓ "در اصل"
      ✗ "گام بردارید"     → ✓ "سپری کنید"
      ✗ "چنان‌که تقریباً بهشتی به نظر می‌رسد" → ✓ "و حالتی تقریباً آسمانی دارد"

   c) DOWNGRADE_INFLATION: if the Persian word is heavier than the
      English, swap to a matching-weight word. Stay within ±10 % of
      the source's emotional load. Equally: do not flatten an
      expressive source (exclamation, intensifier) to neutral.
      ✗ "برای تعالی جهان"          → ✓ "برای بهتر کردن جهان"
      ✗ "مردم تاب‌آور زازا"          → ✓ "مردم مقاوم زازا"
      ✗ "برکات بی‌کران"             → ✓ "برکات فراوان"

   d) NATIVE_CADENCE: replace EN-shaped phrasings with their Persian
      idiomatic counterpart.
      ✗ "از راه این کار، او صبر را می‌آموزد"
      ✓ "این کار به او صبر می‌آموزد"
      ✗ "فیلم صمیمی و زمینی می‌ماند"
      ✓ "فیلم همچنان صمیمی و واقع‌گرا باقی می‌ماند"

   All four sub-rules respect the CONSERVATISM_GATE: if uncertain
   whether the change preserves meaning → ABANDON. Never invent
   structure; only repair when the literal-shaped pattern is visible.

⑦ NUMBER_ROUNDING — round integers expressed as exact multiples of 1,000 or
   1,000,000 collapse to a single word. Examples: ۳٬۰۰۰ → ۳ هزار | ۱۰٬۰۰۰ → ۱۰ هزار |
   ۱۵٬۰۰۰ → ۱۵ هزار | ۲۵٬۰۰۰ → ۲۵ هزار | ۲٬۰۰۰٬۰۰۰ → ۲ میلیون.
   Forbidden: ۱۰۰۰ هزار, ۱ میلیون هزار, ۱۰ هزار هزار.
   Non-round numbers (۱٬۵۰۰ / ۳٬۷۵۰ / ۱۲٬۴۰۰) stay as-is. Never invent a rounding.

⑧ Tighten bloated phrasing — substitution, not deletion. Permitted:
     اقدام به انجام دادن    → انجام داد
     مورد بررسی قرار داد    → بررسی کرد
     مبادرت به اعلام کرد    → اعلام کرد
     به‌شمار می‌رود          → است
     چیزی نگفت              → سکوت کرد
     به پایان رسید          → پایان یافت
     می‌باشد                → است
     می‌گردد                → می‌شود
     می‌نمایند              → می‌کنند
   Tighten connectors only when the shorter form carries identical logical weight:
     به این دلیل که  → چون
     با وجود اینکه   → هرچند
     در صورتی که     → اگر
     به طوری که      → تا
   Drop zero-weight fillers only if meaning is fully preserved:
     در واقع | به طور کلی | در اصل | به نوعی | به نحوی
     Test: read without it — if anything feels missing → keep it.
   Pronoun drop only if antecedent is unmistakably clear on the SAME line.

   FORBIDDEN in this phase:
     • Drop emphasis words: کاملاً | آشکارا | دقیقاً | حداقل | قطعاً | به‌شدت | متأسفانه | خوشبختانه.
     • Reduce intentional repetition (HL-3).
     • Make SAGE phrasing cold or transactional.
     • Change which line any content word appears on.
     • Retranslate: if output differs from [FA] by > 50 % of tokens → revert.
</EDIT>
