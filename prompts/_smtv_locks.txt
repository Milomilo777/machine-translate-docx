# WHITELIST (W1–W5) — frozen tokens
W1) URLs | @handles | #hashtags                                  → BYTE-ID
W2) News-source token in parens at end of a news line            → BYTE-ID
    (Reuters) (AFP) (AP) (Lao Động) (VTV) (Báo Tin Tức)
    SCOPE: only the source token itself; every other (...) and
    [...] on the same line translates normally / BRACKETGLOSS.
    Government / NGO / company names → translate
    (US Department of Labor → وزارت کار آمریکا)
W3) TECH_LOCK codes                                              → BYTE-ID
    version codes (v2.1) | model codes (GPT-4o, F-16)
    formulas (H5N1, NaCl) | serial patterns (S01E05)
    ⚠ NOT TECH_LOCK: vitamin codes (A B C D E K) →
        transliterate: A→آ B→بی C→سی D→دی E→ای K→کا
W4) Placeholders                                                 → BYTE-ID
    {…} | %s/%d | $VAR | <span> | \n | \t | path/to/file | 00:01:23,456
W5) Quoted titles                                                → TRANSLATE FULLY
    Every word of the title must be translated/transliterated. No EN residue.
    Retain " " quotes. FALLBACK only if untranslatable → transliterate.
    EXCEPTION: a complete literary/poetic sentence inside " " is a
    LITERARY_QUOTE, not a title — handle as quote, not as title.

# SMTV COMPOUNDS — spiritual-equality doctrine
INPUT-DRIVEN ONLY. Activate only when input already contains an SMTV
compound (or one in ±10 surrounding lines). Plain animal words on
their own → ordinary translation. Never invent compounds.

Format:  [Honorific]- [Species]   (hyphen + ONE space)
  ✓ شخص- سگ | اشخاص- گاو | شهروند- گربه | نبرد اشخاص- سگ
  ✗ [S]-[H]   |   [H]-[S]   |   [H] - [S]

Honorifics:
  person / being                  → شخص (sg) | اشخاص (pl)
  citizen / folk / people         → شهروند | شهروندان
  friend / companion              → دوست
  resident / inhabitant           → ساکن | ساکنان
  co-inhabitant / fellow-inhabit. → هم‌ساکن (-ان)
  co-citizen / fellow-citizen     → هم‌شهروند (-ان)
  collective / community / group  → اشخاص

Species:  from the compound in the input, or scan ±10 lines.
  Use حیوان only if genuinely unidentifiable.
  FORBIDDEN: skip the honorific, write the species in English,
  pluralise the species, drop the hyphen, or reorder.

SMTV_OVERRIDE_TRUST: silently auto-correct a malformed input
compound to the canonical format above.

# SPIRITUAL TITLES — frozen byte-for-byte
استاد اعظم چینگ های  ← all forms of "(Supreme) Master Ching Hai"
اساتید | استاد روشن‌ضمیر
Masters → respectful + plural verb. God / Allah → reverent + singular verb.

# BRAND / CHANNEL NAMES — fixed Persian forms
The TV channel has a canonical Persian name. Do NOT translate
"Supreme Master" piece-by-piece (wrong forms: "تلویزیون استاد اعظم",
"استاد اعظم تلویزیون"). Use:
  Supreme Master TV         → سوپریم مستر تلویزیون
  Supreme Master Television → سوپریم مستر تلویزیون
  SMTV                      → سوپریم مستر تلویزیون

Channel name ≠ person name. Person = "استاد اعظم چینگ های"; channel
= "سوپریم مستر تلویزیون". Never conflate.

# COMMON_MT_PATTERNS — high-frequency calques + false-friends
# Compiled from 4 Gemini reviews of real SMTV docx (2026-05-13).
# Apply these as bans up front (translator) AND as repair targets
# (polisher). Shared so neither role re-learns the table.

  PASSIVE→ACTIVE (agent is known or implied)
    ستوده می‌شویم            → ما را می‌ستایند
    یافت می‌شوند              → زندگی می‌کنند / دیده می‌شوند
    جاودانه شده‌ایم            → تصویر ما ثبت شده است
    ستوده شده است            → شناخته می‌شود / به جا گذاشته است
    نگریسته می‌شوند           → با ... برخورد می‌شود / ... دارند

  FALSE-FRIEND VERBS
    observe (= state/remark)  → خاطرنشان می‌کند  (نه مشاهده می‌کند)
    desperate (= intensely)    → با تمام وجود     (نه ناامیدانه)
    hailed (= recognised)      → شناخته می‌شود    (نه ستوده شده)
    shave seconds off          → کندتر می‌کند     (نه زمان را کم می‌کند)
    Spanish conquest           → استیلای اسپانیایی‌ها (نه فتح اسپانیا)

  VERB-COLLOCATION CALQUES
    شریک شدن چیزی با کسی     → شریک بودن در چیزی
    گذر زمان را برقرار ساخت   → به جریان انداخت / پدید آورد
    نام را به X گسترش دادند   → نام را به X تعمیم دادند
    مزاحم نمی‌شود              → مزاحمت ایجاد نمی‌کند / توی دست‌و‌پا نیست
    حال‌شان خوب خواهد شد       → خوب می‌شوند / خطر رفع شده

  AWKWARD PERSIANISATIONS (prefer transliteration)
    آسان‌مرگ / آسان‌مرگی        → یوتانایز / یوتانایز کردن
    به خواب ابدی سپرده شدن    → یوتانایز شدن
    شکل‌یافته                  → شکل گرفته‌اند

  GRAMMAR — NEKARE-EZAFE BAN
    Indefinite "ی" before an ezafe phrase is wrong:
      ✗ پدریِ دلسوزِ چهار شخص- گربه
      ✓ پدر دلسوزِ چهار شخص- گربه
      ✗ هم‌خانه‌ای تازه ساکن می‌شود
      ✓ با هم‌خانه‌ی تازه‌ای ... هم‌نشین می‌شود

  SMTV REGISTER MICRO-PATTERNS
    bright viewers (= insightful) → بینندگان فهیم / هوشمند (نه درخشان)
    sensitive souls               → اهل دل / حساس       (avoid "روح" in casual)
    "گرم و پرابراز"               → خونگرم و احساساتی
