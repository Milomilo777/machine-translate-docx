# PROTECTED TOKEN POLICY (W1–W5)
W1–W4 = frozen byte-id. W5 = quoted-title translation policy.

W2) News-source token at end of news line → BYTE-ID

    SCOPE: only the source token. Other (...) and [...] translate normally / BRACKETGLOSS.
    Government/NGO/company names → translate (US Department of Labor → وزارت کار آمریکا).
W3) TECH_LOCK codes → BYTE-ID
    version (v2.1) | model (GPT-4o, F-16) | formula (H5N1, NaCl) | serial (S01E05)
    ⚠ NOT TECH_LOCK: vitamin codes (A B C D E K) → transliterate: A→آ B→بی C→سی D→دی E→ای K→کا
W4) Placeholders → BYTE-ID
    {…} | %s/%d | $VAR | <span> | \n | \t | path/to/file | 00:01:23,456
W5) Quoted titles → TRANSLATE FULLY (not frozen).
    Every word translated/transliterated. No EN residue. Retain " ". Untranslatable fallback → transliterate.
    EXCEPTION: complete literary/poetic sentence, scripture, legal text, song line, attested quotation inside " " = VERBATIM/LITERARY_QUOTE → byte-id including internal punctuation (LS-10 polish).

# SMTV COMPOUNDS — spiritual-equality doctrine
INPUT-DRIVEN ONLY. Activate only when input contains an SMTV compound (or one in ±10 lines). Plain animal words alone → ordinary translation. Never invent compounds.

Format: [Honorific]- [Species] (hyphen + ONE space)
✓ شخص- سگ | اشخاص- گاو | شهروند- گربه | نبرد اشخاص- سگ
✗ [S]-[H] | [H]-[S] | [H] - [S]

Honorifics:
  person/being → شخص (sg) | اشخاص (pl)
  citizen/folk/people → شهروند | شهروندان
  friend/companion → دوست
  resident → ساکن | ساکنان
  co-inhabitant → هم‌ساکن (-ان)
  co-citizen → هم‌شهروند (-ان)
  collective/community → اشخاص

Species: from input compound or scan ±10 lines. Use حیوان only if genuinely unidentifiable.
FORBIDDEN: skip honorific | species in EN | pluralise species | drop hyphen | reorder.

SMTV_OVERRIDE_TRUST: silently auto-correct malformed input compound to canonical format.

FALSE-FRIEND VERBS (context-conditional; literal sense stays when present)
  observe (= state/remark)         → خاطرنشان می‌کند
    observe (= watch/perceive)     → مشاهده می‌کند (literal)
  desperate (= intensely)          → با تمام وجود

  hailed (= publicly recognised)   → شناخته می‌شود

  shave seconds off — strongly context-dependent. Translate by actual sense:
    subtracting from a lifespan/positive → از ... می‌کاهد / کم می‌کند

  Spanish conquest                  → استیلای اسپانیایی‌ها
