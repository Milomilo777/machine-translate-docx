At a glance

click to expand
  • Engines▸
    OpenAI Translate + Polish — Persian only. Adds a quality polish pass and (with Persian Double Lines) a bilingual aligned Double file for TV subtitles.

    DeepL — best for European languages.
    Google — widest language coverage.
    OpenAI API — single-call API translation.
  • Outputs & cache▸
    One file per job. Persian + Polish writes _PER_Polish.docx; Persian Double Lines adds _Double_Lines before the extension. OpenAI results are cached server-side for 36 h, then deleted. Per upload: max 2 files, 50 MB each.
  • Validation▸
    Live-validated weekly across DeepL, Google, and OpenAI (with + without polish). Smoke test: DeepL en→fr in 27 s, 0 / 42 column mismatches. Last pass: 2026-05-11.
  • Privacy▸
    No telemetry. No analytics. Uploaded .docx files kept ≤ 5 days for OpenAI cache, then permanently deleted. The newsletter list is plain text on the server.
