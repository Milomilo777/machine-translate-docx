      How it works

      Drop a Word file on the left, choose target language and engine, then download the result. Works for general docs and broadcast subtitles.

      Engines

      • OpenAI Translate + Polish — Persian only. Adds a quality polish pass and a bilingual aligned Double file for TV subtitles.
      • DeepL — best for European languages.
      • Google — widest language coverage.
      • OpenAI API — single-call API translation.

      Outputs & cache

      • One file per job. Persian + Polish writes _PER_Polish.docx; Persian Double Lines adds _Double_Lines before the extension.
      • OpenAI results are cached server-side for 36 h, then deleted.
      • Per upload: max 2 files, 50 MB each.
